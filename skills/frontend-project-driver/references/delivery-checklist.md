# 交付清单

## 单 agent 路线收尾

- 是否完成目标范围
- 是否覆盖主要验收标准
- 是否明确剩余风险和 tradeoff
- 是否需要进入测试评审或回归评审

## 多 agent 路线收尾

- 每个子任务是否都有 handoff
- handoff 是否足够让下游继续，不需要整包回放上下文
- 是否已完成 `references/review/frontend-testing/SKILL.md` 指定的 testing review
- 是否已完成 `references/review/regression-review/SKILL.md` 指定的 regression review

## 默认质量门槛

- `lint`
- `typecheck`
- `test`
- 关键验收路径可验证

## 最终输出

- 做了什么
- 还缺什么
- 最大风险是什么
- 下一步是继续实现、进入 review，还是可以交付
